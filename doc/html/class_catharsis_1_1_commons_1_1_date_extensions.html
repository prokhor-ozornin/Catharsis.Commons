For a given date/time instance returns a new date/time, representing the start of day.

Parameters
selfOriginal date/time object instance.
Returns
New date/time object instance that represent the start of day of the specified self .

Date component (year, month, day) remains the same, while time component (hour/minute/second) is changed to represent the beginning of the day (hour : 0, minute : 0, second : 0).

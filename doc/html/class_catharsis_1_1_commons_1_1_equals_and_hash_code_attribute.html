Detailed Description

This attribute is used to specify the list of properties of the target class or structure to be used in equality comparison and hash codes calculation.

If present, it's used automatically by the following extension methods:

  • ObjectExtensions.Equality<T>(T, T, string[])
  • ObjectExtensions.Equality<T>(T, T, Expression<Func<T, object>>[])
  • ObjectExtensions.GetHashCode<T>(T, IEnumerable<string>)
  • ObjectExtensions.GetHashCode<T>(T, Expression<Func<T, object>>[])
See also
object.Equals(object), object.GetHashCode(), ObjectExtensions.Equality<T>(T, T, string[]), ObjectExtensions.Equality<T>(T, T, Expression<Func<T, object>>[]), ObjectExtensions.GetHashCode<T>(T, IEnumerable<string>), ObjectExtensions.GetHashCode<T>(T, Expression<Func<T, object>>[])

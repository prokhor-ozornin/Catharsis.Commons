T  other,
params string []  propertiesOrFields

static

Determines whether specified objects are considered equal by comparing values of the given set of properties/fields on each of them.

The following algorithm is used in equality determination:

  • If both self and other are null references, method returns true.
  • If one of compared objects is null and another is not, method returns false.
  • If both objects references are equal (they represent the same object instance), method returns true.
  • If propertiesOrFields set is either a null reference or contains zero elements, EqualsAndHashCodeAttribute attribute is used for equality comparison.
  • If T type does not contain any properties/fields in propertiesOrFields set, object.Equals(object, object) method is used for equality comparison.
  • If T type contains any of the properties/fields in propertiesOrFields set, their values are used for equality comparison according to object.Equals(object) method of both self and other instances.
Template Parameters
TType of objects to compare.
Parameters
selfCurrent object to compare with the second.
otherSecond object to compare with the current one.
propertiesOrFieldsSet of properties/fields whose values are used in equality comparison.
Returns
true if self and other are considered equal, false otherwise.
See also
object.Equals(object), Equality<T>(T, T, Expression<Func<T, object>>[])

Returns a hash value of a given object, using specified set of properties in its calculation.

The following algorithm is used in hash code calculation:

  • If self is a null reference, methods returns 0.
  • If properties set is either a null reference or contains zero elements, EqualsAndHashCodeAttribute attribute is used for hash code calculation.
  • If T type contains any of the properties in properties set, their values are used for hash code calculation according to object.GetHashCode() method. The sum of self 's properties hash codes is returned in that case.
